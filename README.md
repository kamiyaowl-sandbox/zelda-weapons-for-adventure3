## Guardian shieldの縦ライティング

ブレードのライティングを強化するための寸法メモ。2枚で構成することにする。

もともとの厚みは1.4mm

* 表側(敵側)
    * 1.4mm 一切の変更なし
* 裏側(持ち手側)
    * ネジ留め周辺+外周 10mm
        * height=20.0mm LEDテープが立てて通る程度
    * それ以外
        * height1.4mm
